# Dotfiles

These are my dotfiles for Arch Linux with Hpyrland. 

## Programs
* WM: `Hyprland`
* Font: `FiraCode Nerd Font`
* Shell: `zsh` with `zinit` and `tmux`
* Prompt: `pure`
* Terminal emulator: `foot`
* Editor: `neovim`
* Launcher: `tofi`
* System info tool: `fastfetch`
* Browser: `firefox`
* Bar: `waybar`
* System Monitor: `btop++`
* Notification Daemon: `dunst`
