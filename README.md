# dotfiles
These are my current dotfiles for Arch Linux with i3wm as the window manager. 

### Apps used: 
- system monitor: bashtop
- browser: Brave
- music player: cmus
- notification manager: dunst 
- screenshot tool: flameshot
- camera app: guvcview
- wm: i3
- terminal emulator: kitty
- media player: mpv
- compositor: picom 
- file manager: ranger
- pdf viewer: zathura 
- gtk theme: materia dark 
- text editor: neovim
